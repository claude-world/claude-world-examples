# Contributing to Claude World Examples

Thank you for your interest in contributing!

## Ways to Contribute

### 1. Share Your Examples

Have a useful pattern or technique for Claude Code? We'd love to include it!

**To contribute an example:**
1. Fork this repository
2. Create a new file in `/examples` or `/concepts`
3. Follow the existing format (see other files for reference)
4. Submit a Pull Request

### 2. Improve Existing Content

Found a typo? Have a better explanation? PRs are welcome!

### 3. Join the Discussion

For questions, ideas, and discussions:

**[Join our Discord: Claude Code Builders Taiwan](https://discord.gg/claude-code-builders)**

The Discord is the best place to:
- Ask questions about Claude Code
- Share your experiences
- Get help with specific problems
- Discuss new ideas before implementing

## Content Guidelines

### Do Include

- Practical, tested techniques
- Clear explanations with examples
- Code snippets that work
- Before/after comparisons

### Don't Include

- Untested or theoretical approaches
- Vendor-specific content (keep it general)
- Proprietary code or secrets
- Content that violates Anthropic's usage policies

## Format Guidelines

### File Structure

```markdown
# Title

> One-line summary

## The Problem

[Explain what problem this solves]

## The Solution

[Explain the approach]

## Example

[Provide working example]

## Best Practices

[List dos and don'ts]

---

See also: [Related Topic](./link.md)
```

### Code Blocks

- Use syntax highlighting
- Keep examples concise
- Include comments for clarity
